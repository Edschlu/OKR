# reev OKR Dashboard - Q1 FY26

Live TV dashboard and structured data for reev's Q1 FY26 Objectives and Key Results.

## Files

| File | Description |
|---|---|
| `OKR_TV_Dashboard.html` | Auto-refreshing TV dashboard - open in browser, fullscreen |
| `data/Overview.csv` | High-level summary of all 4 objectives |
| `data/Obj_1_GTM_Engine.csv` | Objective 1: GTM Engine - KRs and initiatives |
| `data/Obj_2_Energy_Tariff.csv` | Objective 2: Energy Tariff - KRs and initiatives |
| `data/Obj_3_Upsell_cARR.csv` | Objective 3: Upsell / cARR growth - KRs and initiatives |
| `data/Obj_4_EMS_Growth.csv` | Objective 4: EMS Growth - KRs and initiatives |

## Objectives

| # | Objective | Progress | Status |
|---|---|---|---|
| 1 | Strengthen and accelerate reev's go-to-market engine | 59% | Off Track |
| 2 | Establish reev as a recognised energy tariff provider | 95% | At Risk |
| 3 | Grow value per customer and deepen wallet share | 86% | On Track |
| 4 | Build the most impactful and fastest-growing EMS in our space | 56% | Off Track |

## Usage

Open `OKR_TV_Dashboard.html` directly in a browser. The page auto-refreshes every 5 minutes and cycles through all objectives. Designed for office TV displays.

The `data/` CSVs are the source of truth for KR tracking - update them to keep the dashboard in sync.
